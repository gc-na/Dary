# JAVA의 "do" 명령어: 반복문과 그 사용법

## 개요
JAVA 프로그래밍 언어에서 "do"는 "do-while" 반복문의 시작을 나타내는 키워드로, 특정 조건이 참인 동안 블록 내의 코드를 반복 실행하는 데 사용됩니다. 이 반복문은 최소한 한 번은 실행된다는 특징이 있습니다.

## 문서화
"do-while" 반복문은 JAVA에서 루프를 구현하는 방법 중 하나로, 다음과 같은 구조를 가집니다:

```java
do {
    // 실행할 코드
} while (조건);
```

### 목적
"do-while" 반복문은 코드를 최소한 한 번 실행해야 할 때 유용합니다. 일반적인 "while" 반복문과 달리, "do-while"은 조건을 체크하기 전에 먼저 코드 블록을 실행합니다.

### 사용법
1. **구문**: "do" 키워드로 시작하고, 중괄호 `{}` 안에 실행할 코드를 작성합니다. 그 후 "while" 키워드와 함께 조건을 명시합니다.
2. **조건**: 조건이 참인 동안 반복이 계속됩니다. 조건은 boolean 값을 반환하는 표현식이어야 합니다.

## 예제
다음은 "do-while" 반복문의 기본적인 예제입니다:

```java
int count = 0;

do {
    System.out.println("Count: " + count);
    count++;
} while (count < 5);
```

이 코드는 0부터 4까지의 값을 출력합니다.

## 설명
### 일반적인 함정 및 추가 참고 사항
- **조건 체크**: "do-while" 반복문은 코드 블록을 실행한 후 조건을 평가하기 때문에, 조건이 거짓일지라도 블록 내의 코드가 최소한 한 번 실행됩니다.
- **무한 루프 방지**: 조건이 항상 참인 경우, 무한 루프가 발생할 수 있습니다. 이를 방지하기 위해 루프 내에서 조건을 변화시키는 로직을 반드시 포함해야 합니다.
- **가독성**: "do-while"은 다른 반복문보다 가독성이 떨어질 수 있으므로, 코드의 의도를 명확하게 하기 위해 주석을 활용하는 것이 좋습니다.

## 한 줄 요약
JAVA의 "do" 키워드는 "do-while" 반복문을 시작하며, 코드 블록을 최소한 한 번 실행하고 조건에 따라 반복을 이어갑니다.
